```

3. Create new API key
4. Copy the key (starts with sk-ant-)
```

### Step 2: Update .env File
```
Open file: d:\project\jarvis_desktop\.env
Replace: sk-ant-your-api-key-here
With: sk-ant-YOUR-ACTUAL-KEY-HERE
Save file: Ctrl+S
```

### Step 3: Launch Jarvis!
```
Command: python jarvis_app.py
Or: Double-click run.bat
```

## 🆘 Troubleshooting

### "GUI won't open"
```
1. Run: python test_quick.py
2. Should show system info
3. If error, check dependencies
```

### "Command not working"
```
1. Check spelling
2. Try different phrasing
3. Ensure app is installed
4. Check Jarvis window for error
```

### "No microphone"
```
1. Optional - can still use text
2. Type commands instead
3. Voice is enhancement, not required
```

### "API Key error"
```
1. Get key from console.anthropic.com
2. Double-check it's correct
3. Check .env file saved
4. Restart application
```
